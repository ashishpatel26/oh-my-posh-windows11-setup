Oh My Posh Setup Guide

Detailed instructions for Scoop Users

1. Install Scoop

If you haven't already, install Scoop by running these lines in a non-admin PowerShell window.

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
irm get.scoop.sh | iex

2. Install Oh My Posh

Add the main bucket and install the application.

scoop bucket add main
scoop install main/oh-my-posh

3. Install Nerd Fonts

Icons require a patched font. We recommend Meslo LGS NF.

oh-my-posh font install
Important: After installing, open your Terminal Settings (Ctrl+,), go to Defaults > Appearance, and set "Font face" to MesloLGS NF.

4. Choose Your Theme Path

Choose Option A for built-in themes or Option B for your custom JSON.

Option A: Apply Built-in Theme

1. Open your profile:

notepad $PROFILE

2. Add this line to the file (Replacing jandedobbeleer with your chosen theme):

oh-my-posh init pwsh --config "$($(scoop prefix oh-my-posh))\themes\jandedobbeleer.omp.json" | Invoke-Expression

Option B: Apply Custom Theme

1. Create and Save Theme File
Run this to create a folder safely outside the Scoop directory:

New-Item -Path "$HOME\.mythemes" -ItemType Directory -Force
notepad "$HOME\.mythemes\custom.omp.json"

Paste your custom JSON code into the Notepad window that opens, then Save and Close.

2. Update Profile
Open your profile configuration:

notepad $PROFILE

Delete any existing oh-my-posh lines and paste this exactly:

oh-my-posh init pwsh --config "$HOME\.mythemes\custom.omp.json" | Invoke-Expression

5. Apply Changes

Reload your profile to see the new theme immediately.

. $PROFILE
